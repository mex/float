# Float

## Usage
* round(`number`, `precision`)
* `a + b`: add(`a`, `b`)
* `a - b`: subtract(`a`, `b`)
* `a * b`: multiply(`a`, `b`)
* `a / b`: divide(`a`, `b`)
* `(a + b + c) / 3`: avg(`a`, `b`, `c`)

You can optionally add an options object as the last argument, `divide(1, 3, { precision: 3})` = 0.333
